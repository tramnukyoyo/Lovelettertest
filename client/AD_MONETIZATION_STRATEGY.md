## Part 2: Non-Intrusive Ad Placement (The Golden Rules)

### Rule 1: Never Show Ads During Active Gameplay ‚ùå

**BAD Examples:**
- ‚ùå Ad pops up mid-game
- ‚ùå Banner covers game controls
- ‚ùå Video plays while player is drawing/voting
- ‚ùå Forced full-screen ad between every action

**Why this kills your game:** Players quit immediately. Conversion drops to 0%.

### Rule 2: Show Ads at Natural Break Points ‚úÖ

**GOOD Examples:**
- ‚úÖ After game ends (before returning to lobby)
- ‚úÖ Between rounds (5-second skippable video)
- ‚úÖ In lobby waiting for players
- ‚úÖ Optional "watch ad for bonus" button
- ‚úÖ On results/leaderboard screen

**Why this works:** Players expect a pause. Ads feel natural, not disruptive.

### Rule 3: Frequency Cap (Maximum Ads Per Session)

**Research shows:**
- **Casual games:** 4-6 ads/session acceptable
- **Midcore/Strategic:** 2-3 ads/session acceptable
- **Competitive/Ranked:** 1-2 ads/session acceptable

**Your Games:**
- **Party games (SUSD, DDF, BingoBuddies):** 3-4 ads/session ‚úÖ
- **Strategic (ClueScale, SchoolQuizGame):** 2-3 ads/session ‚úÖ

### Rule 4: Make Ads Skippable

**Best practice:**
- Video ads: 5-second skip button
- Banner ads: X button in corner
- Interstitial ads: "Continue" button after 3 seconds
- Rewarded ads: 100% optional

**Why:** Players feel in control. Reduces frustration dramatically.

```jsx
// components/InterstitialAd.tsx
import React, { useState, useEffect } from 'react';

export const InterstitialAd: React.FC<{ onClose: () => void }> = ({ onClose }) => {
  const [countdown, setCountdown] = useState(5);

  useEffect(() => {
    const timer = setInterval(() => {
      setCountdown((prev) => {
        if (prev <= 1) {
          clearInterval(timer);
          return 0;
        }
        return prev - 1;
      });
    }, 1000);

    return () => clearInterval(timer);
  }, []);

  return (
    <div className="interstitial-overlay">
      <div className="interstitial-content">
        <div className="ad-container">
          {/* AdSense interstitial ad code */}
          <ins
            className="adsbygoogle"
            style={{ display: 'block', width: '100%', height: '250px' }}
            data-ad-client="ca-pub-XXXXXXXXXXXXXXXX"
            data-ad-slot="9876543210"
          />
        </div>

        {countdown > 0 ? (
          <p className="countdown">Continuing in {countdown}...</p>
        ) : (
          <button onClick={onClose} className="close-button">
            Continue to Game
          </button>
        )}
      </div>
    </div>
  );
};
```

## Part 7: Comparison: Ads vs Premium Cosmetics

### Revenue Comparison (500 Active Users)

| Monetization | Setup | Revenue | User Impact |
|-------------|-------|---------|-------------|
| **Ads Only** | Easy (1-2 hours) | $700-1,800/month | Some annoyance |
| **Premium Only** | Hard (20-40 hours) | $500-2,500/month | Zero annoyance (optional) |
| **Both (Hybrid)** | Medium (25-45 hours) | $1,200-4,000/month | Choice: watch ads or buy premium |

### Pros & Cons

**Ads:**
- ‚úÖ Easier to implement (2-3 hours)
- ‚úÖ Revenue from ALL users
- ‚úÖ No payment processing (AdSense handles it)
- ‚ùå Annoys some players
- ‚ùå AdBlocker reduces revenue (~30% of users)
- ‚ùå CPM varies wildly by geography

**Premium Cosmetics:**
- ‚úÖ Zero annoyance (optional)
- ‚úÖ Players who pay WANT to support you
- ‚úÖ More sustainable long-term
- ‚ùå Harder to implement (20-40 hours)
- ‚ùå Revenue from only 5-15% of users
- ‚ùå Need payment processing (Stripe fees)

**Hybrid (Best of Both):**
- ‚úÖ Maximize revenue from all users
- ‚úÖ Players choose: watch ads (free) or buy premium (ad-free)
- ‚úÖ Respects player choice
- ‚ùå More complex to implement
- ‚ùå Need to coordinate ad + payment systems

## Part 9: User Experience Best Practices

### Do's ‚úÖ

1. **Show countdown before ads**
   - "Ad in 3...2...1..."
   - Gives player warning

2. **Make ads skippable**
   - 5-second skip button
   - Players feel in control

3. **Place ads at natural breaks**
   - After round ends
   - In lobby waiting
   - Between games

4. **Frequency cap**
   - Max 3-4 ads per session
   - 1 ad every 5-10 minutes

5. **Offer ad-free option**
   - Premium subscription removes ads
   - Players appreciate choice

6. **Test with friends**
   - Get feedback before launch
   - Adjust placement based on feedback

### Don'ts ‚ùå

1. **Never interrupt gameplay**
   - No mid-game popups
   - No ads covering controls

2. **Don't force unskippable long videos**
   - Max 5 seconds before skip
   - 15-30 second videos ok if skippable

3. **Don't show too many ads**
   - 5+ ads = players quit
   - Respect player time

4. **Don't use misleading ads**
   - No "fake close" buttons
   - No malicious ad networks

5. **Don't place ads over content**
   - No banner covering game UI
   - Keep ads in designated zones

6. **Don't make ads mandatory for features**
   - Rewarded videos = optional only
   - Never lock gameplay behind ads
